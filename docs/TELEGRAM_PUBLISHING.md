# Публикация в Telegram-канал

Платформа публикует одобренные новости в твой Telegram-канал через **Bot API**: бот отправляет сообщения в канал от своего имени.

## Что нужно

1. **Токен бота** (`TELEGRAM_BOT_TOKEN`)  
2. **ID канала** (`TELEGRAM_PUBLISH_CHANNEL_ID`)  
3. **Права бота в канале** — бот должен быть **администратором** канала с правом «Публиковать сообщения».

---

## 1. Создать бота и получить токен

1. Открой [@BotFather](https://t.me/BotFather) в Telegram.  
2. Отправь `/newbot`, придумай имя и username (например `@MyNewsPublisherBot`).  
3. BotFather пришлёт **токен** вида `123456789:AAH...`.  
4. Скопируй его в `.env`:
   ```env
   TELEGRAM_BOT_TOKEN=123456789:AAH...
   ```

---

## 2. Добавить бота в канал и дать права

Бот может писать в канал **только если он администратор** с правом публиковать сообщения.

1. Открой свой канал в Telegram.  
2. Зайди в **Управление каналом** → **Администраторы** → **Добавить администратора**.  
3. Найди своего бота по username (например `@MyNewsPublisherBot`) и добавь.  
4. В правах включи минимум:
   - **Публиковать сообщения** (Post messages)  
   Остальные права для простой публикации не обязательны.  
5. Сохрани.

После этого бот имеет доступ к каналу и может в него постить.

---

## 3. Узнать ID канала

ID канала нужен для переменной `TELEGRAM_PUBLISH_CHANNEL_ID`.

### Вариант A: Публичный канал с @username

Если у канала есть username (например `@mychannel`), можно указать его в `.env`:

```env
TELEGRAM_PUBLISH_CHANNEL_ID=@mychannel
```

### Вариант B: Приватный канал или числовой ID

Числовой ID канала имеет вид `-100xxxxxxxxxx` (например `-1001234567890`).

Как получить:

1. Перешли в канал любое сообщение от бота (или добавь бота в канал и перешли его сообщение).  
2. Открой в браузере:  
   `https://api.telegram.org/bot<ТВОЙ_ТОКЕН>/getUpdates`  
   (подставь свой `TELEGRAM_BOT_TOKEN`).  
3. В ответе найди `"chat":{"id":-1001234567890,...}` — это ID канала.  
4. В `.env` укажи:
   ```env
   TELEGRAM_PUBLISH_CHANNEL_ID=-1001234567890
   ```

Либо используй ботов вроде [@userinfobot](https://t.me/userinfobot) или [@getidsbot](https://t.me/getidsbot): перешли им сообщение из канала — они покажут ID чата (канала).

---

## 4. Итог в `.env`

```env
TELEGRAM_BOT_TOKEN=123456789:AAH...
TELEGRAM_PUBLISH_CHANNEL_ID=@mychannel
# или для приватного канала:
# TELEGRAM_PUBLISH_CHANNEL_ID=-1001234567890
```

Перезапусти бэкенд (и воркеры Celery, если публикация идёт через них). После одобрения статьи в разделе «Модерация» и выбора цели «Telegram» задача отправит сообщение в канал.

---

## Как это устроено в коде

- В админке при публикации статьи выбираются цели: **Сайт** и/или **Telegram**.  
- Для Telegram создаётся запись в БД и ставится задача Celery `publish_to_telegram`.  
- Воркер вызывает Telegram Bot API:  
  `POST https://api.telegram.org/bot<TOKEN>/sendMessage`  
  с `chat_id` = твой канал и текстом (заголовок, краткое описание, теги в HTML).  
- Если токен или ID канала не заданы, в логах будет «Telegram bot token or channel ID not configured» и публикация в Telegram не выполнится.
